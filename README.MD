# Campo Minado (Terminal)

Projeto desenvolvido em Java com o objetivo de recriar o clássico jogo Campo Minado para execução diretamente no terminal.

## 📋 Sobre o Projeto

Este projeto foi criado para praticar conceitos de programação orientada a objetos, estruturas de dados e lógica de programação utilizando Java.

O jogo permite que o usuário interaja com um tabuleiro através do terminal, marcando posições e revelando campos enquanto tenta evitar as minas escondidas.

## 🚀 Funcionalidades

- Criação do tabuleiro
- Distribuição de minas
- Abertura de campos
- Marcação e desmarcação de bandeiras
- Verificação de vitória e derrota
- Exibição do estado atual do jogo no terminal

## 🛠️ Tecnologias Utilizadas

- Java
- Programação Orientada a Objetos (POO)

## 📂 Estrutura do Projeto

O projeto está organizado em classes responsáveis por representar:

- O tabuleiro
- Os campos
- A lógica do jogo
- A interação com o usuário

## 🎯 Objetivos de Aprendizagem

Durante o desenvolvimento deste projeto foram praticados conceitos como:

- Classes e Objetos
- Encapsulamento
- Collections
- Tratamento de Exceções
- Estruturas de Repetição
- Estruturas Condicionais
- Manipulação de Entrada e Saída

## ▶️ Como Executar

1. Clone o repositório:

```bash
git clone https://github.com/JoaoZ4/Campo-Minado.git
```

2. Acesse a pasta do projeto:

```bash
cd Campo-Minado
```

3. Compile e execute o projeto utilizando sua IDE Java preferida ou através do terminal.
